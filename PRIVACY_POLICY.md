# Privacy Policy

**Last Updated: January 2024**

## Overview

Lovable Voice Helper ("the Extension") is a Chrome browser extension that enables voice input for Lovable.dev. This privacy policy explains how we collect, use, and protect your information.

## Information We Collect

### Account Information
When you sign in with Google, we collect:
- Email address
- Display name
- Profile picture URL

This information is used to identify your account and personalize your experience.

### Voice Recordings
When you use the voice input feature:
- Audio is recorded locally in your browser
- Audio is transmitted to our servers for transcription
- Audio is processed by Google's Gemini API
- **Audio is NOT stored permanently** - it is deleted after transcription

### Transcribed Text
We store:
- The original transcribed text (in your language)
- The translated English text
- The elaborated prompt
- The language detected
- Selected objective type
- Any additional context you provide

This is used for:
- Processing your requests
- Usage analytics
- Improving our service

### Usage Data
We track:
- Number of voice prompts per day
- Timestamp of each request
- Audio duration (seconds)
- Text lengths (characters)
- Token usage

This is used for:
- Enforcing usage limits
- Service analytics
- Billing (for paid plans)

## How We Use Your Information

1. **Service Delivery**: Processing your voice input and generating prompts
2. **Account Management**: Authentication and subscription management
3. **Usage Limits**: Tracking daily usage against your plan limits
4. **Analytics**: Understanding how the service is used to improve it
5. **Support**: Helping resolve issues if you contact us

## Third-Party Services

### Google Gemini
- Audio is sent to Google's Gemini API for transcription
- Text is sent to Google's Gemini API for translation and elaboration
- Google's AI privacy policy applies: https://ai.google.dev/terms

### Google
- We use Google OAuth for authentication
- Google's privacy policy applies: https://policies.google.com/privacy

### Supabase
- User data is stored on Supabase infrastructure
- Supabase's privacy policy applies: https://supabase.com/privacy

## Data Storage

- Data is stored in Supabase databases
- Data is encrypted in transit (HTTPS/TLS)
- Data is encrypted at rest
- Data is stored in secure cloud infrastructure

## Data Retention

- Account data: Retained while your account is active
- Usage logs: Retained for 90 days
- Audio recordings: Deleted immediately after transcription

## Your Rights

You have the right to:
- **Access**: Request a copy of your data
- **Delete**: Request deletion of your account and data
- **Export**: Download your usage history
- **Opt-out**: Stop using the service at any time

To exercise these rights, contact: support@lovable-voice-helper.com

## Data Security

We implement security measures including:
- HTTPS encryption for all data transmission
- Secure authentication via Google OAuth
- Row-level security on database tables
- Regular security reviews
